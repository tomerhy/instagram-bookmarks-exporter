Privacy Policy

Last updated: February 2026

Overview

Instagram Saved Media Exporter is a browser extension that helps you export images and videos from your saved Instagram posts. We are committed to protecting your privacy.

Data Collection

This extension does NOT collect, store, or transmit any personal data to external servers. All data processing happens locally in your browser.

What the Extension Does

  • Intercepts media URLs (images and videos) from Instagram pages you visit
  • Stores these URLs locally in your browser's storage
  • Provides a gallery interface to view and download captured media

Data Storage

Media URLs are stored using Chrome's local storage API (chrome.storage.local). This data remains on your device and is never sent to any external server.

Permissions

The extension requires the following permissions:

  • storage: To save captured media URLs locally
  • tabs: To detect when you're on Instagram
  • scripting: To inject content scripts for media detection
  • Host permissions for instagram.com: To capture media from Instagram

Third-Party Services

This extension does not use any third-party analytics, tracking, or data collection services.

Changes to This Policy

We may update this privacy policy from time to time. Any changes will be reflected on this page.
